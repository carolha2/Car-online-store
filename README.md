## About The Project

Website to help car sellers organize their inventory and sales.

#### Concepts:
* Transactions (Optimistic Lock)
* Testing (Integration Test)
* Caches
* Aspect Logging

#### Architecture:
* Web aplication
* RESTful API
* ORM
* JWT Authentication
* JWT Authorization
